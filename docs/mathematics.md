# Mathematical Notes

Free reduction cancels adjacent inverse pairs and preserves the represented free-group element. Cyclic reduction is not automatic because it changes a relator by conjugation; in AC-Zero that must be represented by explicit `AC3` primitive moves.

The default goal is the exact standard presentation `<x1,...,xn | x1,...,xn>`. The signed-permuted-basis mode is separated because cleanup moves must be explicit and verified.

A failed search is not mathematical evidence that an input is a genuine counterexample. Only a verified strict primitive certificate supports a claim that the specific presentation has been trivialized.
